2.3. Tính F-mearure của kết quả gom cụm dùng true label¶

Gi s tp d liu $D$ c $n$ phn t $x_i$ c phn hoch thnh $p$ nhm ( y ng vi s loi). Gi $y_i \in \{1, 2, , p\}$ l nhm tht s (ground-truth labels) cho mi phn t. Ground-truth clustering c cho bi $T = \{T_1, T_2, \cdots , T_p\}$, vi $T_j$ bao gm tt c cc phn t c nhn $j$, ngha l, $T_j = \{ x_i \in D | y_i = j \}$. Mt khc, gi $C = \{ C_1, C_2, \cdots, C_k \}$ l mt kt qu gom cm ca $D$ thnh $k$ cm (cluster), qua mt thut ton gom cm no , v $\hat{y_i} \in \{ 1, 2, \cdots, k \}$ l cluster label cho $x_i$. Ta s xem $T$ l mt phn hoch chun (ground-truth partitioning) v mi $T_i$ l mt phn vng (partition). Ta gi $C$ l mt kt qu gom cm (clustering), vi mi $C_i$ l mt cm (cluster). Gi s ground truth l bit trc, mt thut ton gom cm s thc hin gom cm trn $D$ vi s cm chnh xc, tc vi $k = p$. Tuy nhin, gi tnh tng qut, ta cho php $k \ne p$.

Clusters/Species $T_1$ $T_2$ $\cdots$ $T_p$
$C_1$ $n_{11}$ $n_{12}$ $\cdots$ $n_{1p}$
$C_2$ $n_{21}$ $n_{22}$ $\cdots$ $n_{2p}$
$\vdots$ $\vdots$ $\vdots$ $\ddots$ $\vdots$
$C_k$ $n_{k1}$ $n_{k2}$ $\cdots$ $n_{kp}$

Cc o nh gi kt qu gom cm c gng nm bt mc m cc phn t t cng mt phn vng (partition) xut hin trong cng mt cm (cluster) v mc m cc phn t t cc phn vng (partition) khc nhau c nhm thnh cc cm (cluster) khc nhau. Nhng o ny da trn $k \times p$ contingency table $N$ (xem bng trn) c thnh lp da vo mt kt qu gom cm (clustering) $C$ v mt phn hoch chun (ground-truth partitioning) $T$, c nh ngha nh sau:

$$N(i, j) = n_{ij} = |C_i \cap T_j|$$

  • $Recall$ l t l i tng cng loi c gn cng cm.

  • $Precision$ l t l i tng c gn cng cm thuc cng loi.

  • $F{\text -}measure$ l mt o cn bng gia $Precision$ v $Recall$ v c tnh bng trung bnh iu ha gia $Precision$ v $Recall$. y l mt o thng c s dng so snh cc thut ton gom cm vi nhau.

Cc o $Precision$, $Recall$, v $F{\text -}measure$ c tnh t contingency table trn dng cc cng thc sau:

\begin{equation} precision = \frac{{\sum\limits_{i = 1}^k {\mathop {{\rm{max}}}\limits_{j \in \left\{ {1, \ldots p} \right\}} \{n_{ij}\} } }}{{\sum\limits_{i = 1}^k {\sum\limits_{j = 1}^p {n_{ij} } } }} \end{equation}\begin{equation} recall = \frac{{\sum\limits_{j = 1}^p {\mathop {{\rm{max}}}\limits_{i \in \left\{ {1, \ldots ,k} \right\}} \{n_{ij}\} } }}{{\sum\limits_{i = 1}^k {\sum\limits_{j = 1}^p {n_{ij} } } }} \end{equation}\begin{equation} \begin{split} F{\text -}measure = \frac{{2 \cdot precision \cdot recall}}{{precision + recall}} \end{split} \end{equation}

from sklearn.metrics.cluster import contingency_matrix
import numpy as np

def precision(labels_true, labels_pred):
    cm = contingency_matrix(labels_true, labels_pred)
    t = np.sum(np.max(cm, axis=0))
    m = np.sum(cm)
    return t/m


def recall(labels_true, labels_pred):
    cm = contingency_matrix(labels_true, labels_pred)
    t = np.sum(np.max(cm, axis=1))
    m = np.sum(cm)
    return t/m


def fmeasure(labels_true, labels_pred):
    a = precision(labels_true, labels_pred)
    b = recall(labels_true, labels_pred)
    return (2*a*b)/(a+b)

# Vit code ca bn  y
import numpy as np
from matplotlib import pyplot as plt
from scipy.cluster.hierarchy import dendrogram

from sklearn.cluster import AgglomerativeClustering
from sklearn.datasets import load_iris

def plot_dendrogram(model, **kwargs):
    # Create linkage matrix and then plot the dendrogram

    # create the counts of samples under each node
    counts = np.zeros(model.children_.shape[0])
    n_samples = len(model.labels_)
    for i, merge in enumerate(model.children_):
        current_count = 0
        for child_idx in merge:
            if child_idx < n_samples:
                current_count += 1  # leaf node
            else:
                current_count += counts[child_idx - n_samples]
        counts[i] = current_count

    linkage_matrix = np.column_stack(
        [model.children_, model.distances_, counts]
    ).astype(float)

    # Plot the corresponding dendrogram
    dendrogram(linkage_matrix, **kwargs)


iris = load_iris()
X = iris.data

# setting distance_threshold=0 ensures we compute the full tree.
model = AgglomerativeClustering(distance_threshold=0, n_clusters=None)

model = model.fit(X)
plt.title("Hierarchical Clustering Dendrogram")
# plot the top three levels of the dendrogram
plot_dendrogram(model, truncate_mode="level", p=3)
plt.xlabel("Number of points in node (or index of point if no parenthesis).")
plt.show()

Thc li cc bc mc 3 trn tp d liu iris vi thut ton BisectingKmeans. BisectingKmeans l mt thut ton thuc nhm hierarchical clustering v theo cch tip cn top-down.

Tham kho:

  1. BisectingKmeans User Guides

  2. BisectingKmeans API Documentations

Bisecting Kmeans l mt thut ton gm cm thuc nhm hierarchical clustering. Khc vi thut ton AgglomerativeClustering tip cn theo hng bottom-up, thut ton Bisecting Kmeans tip cn theo hng top-down.

tng ca thut ton Bisecting Kmeans

Thut ton khi u bng vic xem tt c cc im d liu thuc cng mt cm v sau tin hnh nh sau:

  1. Chn mt cm tch
  2. Tch cm chn thnh 2 sub-clusters dng thut ton K-means. (bisecting step)
  3. Lp li bc 2 (bisecting step) iter ln v chn ra php tch to ra cc cm c tng t tng th cao nht.
  4. Lp li bc 1, 2, v 3 cho n khi t c s cm mong mun.

  • C nhiu cch khc nhau chn cm tch (bc 1). Ta c th chn cm c kch c ln nht (nhiu phn t nht) mi bc, hoc chn cm c tng t tng th nh nht, hoc chn cm dng c hai tiu ch v kch c v tng t tng th.

  • tng t tng th (overall similarity) ca mt cm c tnh bng tng t trung bnh ca cc cp phn t trong cng mt cm (pairwise similarity).
